## ✍️ 나만의 독서록 작성 규칙 (Rules)

책을 단순히 읽고 끝내지 않기 위해 아래 규칙을 지켜 기록합니다.

1. **핵심 요약:** 책의 핵심 메시지를 3문장 이내로 요약하기.
2. **이펙트 (Effect):** 책을 읽고 새롭게 알게 된 점이나 깨달은 점 적기.
3. **액션 아이템 (Action Item):** 내 삶이나 코드에 적용할 구체적인 행동 1가지 이상 도출하기.
